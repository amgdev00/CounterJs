🧮 Interactive Counter

A simple and interactive counter application built with HTML, CSS, and JavaScript. This project demonstrates basic DOM manipulation, event handling, and clean UI structure for beginners.

Responsive and centered layout
Clean and minimal UI

📌 How It Works

The application is structured around a main counter display and three buttons:

Increase Button (+1) → Adds 1 to the current value
Decrease Button (-1) → Subtracts 1 from the current value
Reset Button → Sets the counter back to 0

The JavaScript uses addEventListener to detect user interactions and update the DOM dynamically.
