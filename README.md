## 🚀 Features

| Feature | Description |
|---|---|
| ➕ Increase | Adds 1 to the counter |
| ➖ Decrease | Subtracts 1 from the counter |
| 🔄 Reset | Returns the counter to zero |
| 🎨 Clean UI | Centered, minimal and responsive layout |
| 🪶 Lightweight | No frameworks, no dependencies |

## 📌 How It Works

JavaScript listens to button clicks and dynamically updates the counter value using **DOM manipulation** — no libraries required.

[➕ Increase]  →  counter + 1

[➖ Decrease]  →  counter - 1

[🔄 Reset   ]  →  counter = 0
